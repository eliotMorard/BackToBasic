Module 2 : Manipulations en C pour les Microcontrôleurs

Découvrez les bases des opérations en C pour configurer et manipuler vos microcontrôleurs.

Opérations sur les bits : Les bases indispensables

En programmation C, les opérations sur les bits sont essentielles pour manipuler directement les registres ou configurer des périphériques matériels. Ces opérations permettent d'activer, désactiver ou basculer des bits spécifiques dans un registre. Voici les trois opérations fondamentales et leur utilisation typique :

1. Mettre un bit à 1

          registre |= (1 << position);  // Active le bit à la position spécifiée

Cette opération utilise l'opérateur OR bit à bit (|=) pour activer un bit à une position donnée. Par exemple, si position = 3, le troisième bit du registre sera mis à 1, sans affecter les autres bits.

2. Mettre un bit à 0

          registre &= ~(1 << position);  // Désactive le bit à la position spécifiée

Ici, l'opérateur AND bit à bit (&=) est combiné avec un complément (~) pour désactiver un bit spécifique. Par exemple, si position = 3, le troisième bit sera mis à 0, tandis que les autres bits restent inchangés.

3. Basculer un bit

          registre ^= (1 << position);  // Inverse l'état du bit à la position spécifiée

L'opérateur XOR bit à bit (^=) permet d'inverser l'état d'un bit. Si le bit est 0, il devient 1 ; s'il est 1, il devient 0. Par exemple, si position = 3, le troisième bit du registre sera inversé.

Manipulation des registres en C

Après avoir compris les bases des opérations sur les bits, nous pouvons maintenant aborder la manipulation directe des registres en langage C. Ces opérations sont fondamentales pour configurer les périphériques matériels et interagir avec un microcontrôleur. 

        LPC_GPIO_PORT->DIR0 |= (1 << 10);  // Configure la broche P0_10 comme sortie

Cet exemple montre comment configurer une broche GPIO (General Purpose Input/Output) pour qu’elle fonctionne comme une sortie. En activant le bit correspondant dans le registre DIR0, vous indiquez au microcontrôleur que cette broche est utilisée pour envoyer des signaux.

Manipulation des bits : Tutoriels pratiques

Les microcontrôleurs utilisent des registres pour configurer et contrôler leur fonctionnement. Ces registres sont souvent manipulés à l’aide d’opérations bit à bit, comme définir un bit à 1, le mettre à 0, ou inverser son état. Voici quelques tutoriels pour maîtriser ces opérations.

1. Mettre un bit à 1

Pour forcer un bit spécifique à 1 sans modifier les autres bits, on utilise l’opération OR (|) avec un masque. 

      // Exemple : Mettre le bit 3 à 1
#define BIT_3 (1 << 3)
registre |= BIT_3;  // Met le bit 3 à 1 sans affecter les autres

2. Mettre un bit à 0

Pour forcer un bit spécifique à 0, on utilise l’opération AND (&) avec l’inverse d’un masque. 

      // Exemple : Mettre le bit 3 à 0
#define BIT_3 (1 << 3)
registre &= ~BIT_3;  // Met le bit 3 à 0 sans affecter les autres

3. Inverser un bit

Pour inverser l’état d’un bit spécifique, on utilise l’opération XOR (^). 

      // Exemple : Inverser le bit 3
#define BIT_3 (1 << 3)
registre ^= BIT_3;  // Inverse l'état du bit 3

4. Appliquer un masque pour extraire des bits

Pour extraire un groupe de bits, on utilise un masque avec l’opération AND (&), puis on décale les bits extraits à droite si nécessaire. 

      // Exemple : Extraire les bits 4 à 6
#define MASK (0b01110000)
int valeur = (registre & MASK) >> 4;  // Extrait et décale les bits 4 à 6

5. Ajuster dynamiquement plusieurs bits

En combinant plusieurs opérations, on peut activer et désactiver des bits spécifiques dynamiquement. 

      // Exemple : Activer les bits 2 et 4
#define CONFIG_MASK ((1 << 2) | (1 << 4))
registre |= CONFIG_MASK;  // Active les bits 2 et 4

// Désactiver les bits 2 et 4
registre &= ~CONFIG_MASK; // Désactive les bits 2 et 4
